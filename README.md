# FIRE 计算器

一个使用 Vue 3 开发的财务自由提前退休 (FIRE) 计算器，帮助你计算当前资金可以支撑的退休年数。

## 功能特点

### 核心功能
- 计算当前资金可以 FIRE 到多少年
- 考虑通胀率和年化收益率的影响
- 显示资金随年龄的变化趋势
- 提供详细的年度资金明细

### 参数设置
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 当前资金量 | 100万 | 可用于投资的资产总额 |
| 当前年龄 | 30岁 | - |
| FIRE年龄 | 45岁 | 计划退休年龄 |
| 月支出 | 8000元 | FIRE后每月预期支出 |
| 通胀率 | 3% | 可修改的通胀率 |
| 年化收益率 | 5% | 可修改的投资回报率 |
| 安全提取率 | 4% | 经典4%法则 |

### 技术特点
- 使用 Vue 3 Composition API 开发
- 数据持久化到 localStorage，刷新页面不丢失
- 现代简洁的界面设计
- 响应式布局，支持移动设备

## 快速开始


## 计算逻辑

1. **积累期**：从当前年龄到 FIRE 年龄，资金按年化收益率增长
2. **消耗期**：从 FIRE 年龄开始，考虑通胀和收益消耗资金
3. **通胀调整**：年支出随通胀率逐年增加
4. **收益计算**：资金按年化收益率增长

## 数据持久化

所有参数会自动保存到浏览器的 localStorage，刷新页面后数据不会丢失。
